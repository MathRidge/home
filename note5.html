Fraction Shelves & Reduction

A fraction is one amount with two shelves. Read the shelves first. Then use the Split Shelf skill when a shared group creates a large division step.

1. Read shelvesTop tells the pieces we have.
2. Share groupTop and bottom use the same group.
3. Clean nameStop when no shared group remains.

01 Read the Fraction as One Amount

Do not rush into reducing. First, name what the fraction means. The top and bottom work together.

One fraction. One amount.

Focus cue: a fraction is not two separate problems. Keep the shelves connected.

Different fraction names can show the same amount. Reduction changes the name, not the value.

05 Quick Group Checks

Use one check at a time. This keeps the page calm and prevents guessing.

2Even?

Both shelves are even.

3Digit sum?

Both digit sums divide by 3.

5Ends in 0 or 5?

Both shelves end in 0 or 5.
