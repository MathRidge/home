Fractions Are Sharing

A fraction is not here to scare us. A fraction tells us how much we have compared to the whole amount. First, we learn to read the top shelf and bottom shelf before we try to calculate anything difficult.

Top shelfHow many pieces we have
Bottom shelfHow many pieces in the whole
Same amountDifferent names can match

1. One Whole Can Still Be Shared

Alex has 1 box of Pringles. Alex says, “I cannot share because I only have 1 box.” But that is not really true. The box is 1 whole, but inside the box there are many chips.

1 whole box
50 chips inside

The important idea

One whole can be split into equal pieces. When we talk about fractions, we are talking about how many pieces we have out of the whole amount.

Fraction thinking starts with sharing, not decimals.

2. Half and Half

If Alex and Beth share 50 chips equally, each of them gets 25 chips. The whole tray has 50 chips. The blue chips show the 25 chips Alex has.

25 out of 50 chips

The whole tray has 10 rows of 5 chips. The blue rows show the 25 chips Alex has.

Blue: 25 chips · Whole: 50 chips

Group each row into 1 stack

Each row of 5 chips becomes 1 group. Now we can read the picture as: 5 groups out of 10 groups.

5 groups out of 10 groups

Split the 10 groups into 2 equal halves

Five groups make the left half. Five groups make the right half. Alex has 1 half out of 2 halves.

Alex has 1 group of five
Beth has 1 group of five
Alex have 1 group of five out of 2 equal groups.
2550 → 5 rows of 510 rows of 5 → 1 group of 52 groups of 5 → 12
Read this as 25 out of 50. After grouping by 5, it becomes 5 out of 10. Then the 10 groups can be seen as 2 equal halves, so 5 groups out of 10 groups is 1 out of 2.

3. Top Shelf and Bottom Shelf

Instead of rushing into big math words, we first read the fraction like a shelf. The top shelf and bottom shelf each have a job.

Top shelf: 3

310

The top shelf tells how many pieces we have.

Bottom shelf: 10

310

The bottom shelf tells how many equal pieces make the whole.

Main reading pattern: top out of bottom. So 3/10 means 3 out of 10.

4. Different Names, Same Amount

Fractions can look different but still mean the same amount. This is like money: half a dollar, 2 quarters, and 50 pennies can all mean the same value.

12 → 24 → 50100 → 2550
These are different names for the same amount: one-half.

5. How to Spot Group Sizes Quickly

Before we reduce a fraction, we need to ask: Can the top shelf and bottom shelf be grouped the same way? In this note, we will focus on how to recognize if a number can be grouped evenly by 2, 3, and 5.

Group by 2

Check the last digit. If the number ends with an even digit, it can be grouped evenly by 2.

02468
24 ends in 4 → can group by 2
38 ends in 8 → can group by 2
35 ends in 5 → cannot fully group by 2

Group by 3

Add the digits. If the digit sum can be grouped by 3, then the whole number can be grouped by 3.

27 → 2 + 7 → 9 → can group by 3
42 → 4 + 2 → 6 → can group by 3
25 → 2 + 5 → 7 → cannot fully group by 3

Group by 5

Check the last digit. If the number ends with 0 or 5, it can be grouped evenly by 5.

05
35 ends in 5 → can group by 5
120 ends in 0 → can group by 5
42 ends in 2 → cannot fully group by 5
If both shelves pass the same grouping check, try that group size on both shelves. If only one shelf passes, do not use that group size yet.

6. Reduction Without Changing the Amount

When the top shelf and bottom shelf can be grouped the same way, we can reduce the fraction form. Reduction does not change the amount. It only changes the fraction name.

Start with the sharing fraction.
2550
Both shelves end in 5 or 0, so both shelves can be grouped by 5.
2550 → 5(5)10(5) → 510
The new shelves are 5 and 10. Both can still be grouped by 5.
510 → 1(5)2(5) → 12
Read the reduction path.
2550 → 510 → 12
We did not change the amount. We only changed the fraction name: 25/50 is same as 5/10 is same as 1/2.

7. Fractions Can Also Mean Division

A fraction can also be read as division. Notice that the division symbol has a dot on the top and a dot on the bottom. We can think of those dots as empty spaces for numbers: top number divided by bottom number becomes top shelf over bottom shelf.

Division symbol way

12 ÷ 3

The top blank is 12. The bottom blank is 3.

Fraction way

123

So 12 divided by 3 can be written as 12 over 3.

123 → 4(3)1(3) → 41 → 4
The 1 on the bottom shelf can be hidden. 4 out of 1 means 4 individual units, so we usually just write 4.

8. Rewriting Into Groups

This connects back to "Chunking Repeated Values". We can rewrite a value into groups first. If the top shelf and bottom shelf have the same group size, we can drop the matching group size and keep the reduced fraction name.

Use group size 2

Both shelves end with even digits, so try grouping by 2.

1824 → 9(2)12(2) → 912

Then use group size 3

9 can group by 3. For 12, add the digits: 1 + 2 → 3, so 12 can group by 3 too.

912 → 3(3)4(3) → 34

Use group size 5

Both shelves end in 0 or 5, so both shelves can group by 5.

1535 → 3(5)7(5) → 37
Notice the pattern: first recognize a group size, then rewrite both shelves using the same group size. Reduction is not guessing. Reduction is looking for a matching group.

9. Practice

For each fraction, choose a group size of 2, 3, or 5. Then rewrite both shelves into matching groups and reduce the fraction name.

1824 can first be grouped by ___.
Answer18/24 → 9(2)/12(2) → 9/12. Then 9/12 → 3(3)/4(3) → 3/4.
2550 can first be grouped by ___.
Answer25/50 → 5(5)/10(5) → 5/10. Then 5/10 → 1(5)/2(5) → 1/2.
2130 can first be grouped by ___.
Answer21/30 → 7(3)/10(3) → 7/10.
2745 can first be grouped by ___.
Answer27/45 → 9(3)/15(3) → 9/15. Then 9/15 → 3(3)/5(3) → 3/5.
4060 can first be grouped by ___.
Answer40/60 → 8(5)/12(5) → 8/12. Then 8/12 → 4(2)/6(2) → 4/6. Then 4/6 → 2(2)/3(2) → 2/3.
